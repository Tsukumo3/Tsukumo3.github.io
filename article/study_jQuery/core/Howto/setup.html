jQuery Get Started
Adding jQuery to Your Web Pages

jQueryを導入するにはいくつかの方法があります。

  1. jQuery LibraryをjQuery.comからダウンロードする
  2. jQuery.comやGoogleのようなCDNからインクルードする
1.Downloading jQuery

ダウンロードできるjQueryのバージョンは2つあります。

  • 製品版-縮小および圧縮されているため、ライブWebサイト用です。
  • 開発バージョン-これはテストおよび開発用です（非圧縮で読み取り可能なコード）

どちらのバージョンもjQuery.comからダウンロードできます。

jQueryライブラリは単一のJavaScriptファイルであり、HTML <script>タグで参照します（<script>タグは<head>セクション内にある必要があります）。


<head>
  <script src="jquery-3.5.1.min.js"></script>
</head>
2.jQuery CDN

自分でjQueryをダウンロードしてホストしたくない場合は、CDN（コンテンツ配信ネットワーク）からjQueryを含めることができます。

jQuery CDNを使う場合


<head>
  <script src="https://code.jquery.com/jquery-3.1.1.min.js"></script>
</head>

Google CDNを使う場合


<head>
  <script src="https://ajax.googleapis.com/ajax/libs/jquery/3.5.1/jquery.min.js"></script>
</head>
